💡 モニター選び方ガイド
🎮 ゲーミング用
推奨：24～27インチ / 144Hz以上 / 1ms以下応答時間
FHD (1920×1080) 推奨。高リフレッシュレートが重要。27インチなら WQHD (2560×1440) で解像度とのバランスが取れます。応答時間が重要なFPS、格闘ゲームは24インチ、ストーリーゲームは27インチがおすすめ。
🎬 動画・画像編集用
推奨：27～32インチ / 4K (3840×2160) / 色精度 Delta E < 2
色再現性が重要。27インチ 4K が標準。32インチは作業エリアが広い。正確な色管理が必要な場合は 5K (5120×2880) も選択肢。IPS パネル推奨。
💼 事務作業・プログラミング用
推奨：27～34インチ / WQHD (2560×1440) ～ 4K / ウルトラワイド推奨
ウルトラワイド (34インチ / 3440×1440) でデュアルモニター相当の作業効率。通常の 27～32インチ WQHD でも十分。長時間作業はブルーライト軽減機能を確認。
🖼️ 画像・グラフィック制作用
推奨：32インチ / 4K～5K / Adobe RGB 99% 以上
広色域が必須。27インチ 4K より 32インチの方が作業面積が広い。USB-C 給電対応で接続がシンプル。
📚 用語解説
インチ（対角線サイズ）

モニターの対角線を inch（インチ）で表示。1インチ = 2.54cm。一般的には22～27インチが主流。

アスペクト比

16:9 - 最も一般的。テレビ・映画と同じ。
16:10 - 少し高い。オフィス作業向け。
21:9（ウルトラワイド） - 横に長い。映画的。マルチウィンドウ作業に最適。
32:9（スーパーウルトラワイド） - さらに横長。デュアルモニター相当。

解像度（ピクセル数）

FHD (1920×1080) - 最も一般的。テキストが少し大きめ。
WQHD (2560×1440) - 標準～高解像度。27インチの最適解像度。
4K (3840×2160) - 高解像度。映像編集に人気。テキストが小さい。
5K (5120×2880) - 超高解像度。プロ向け。高性能 GPU が必要。
8K (7680×4320) - 最上位。future-proof だが普及はまだ先。

PPI（ピクセル・パー・インチ）

1インチあたりのピクセル数。値が高いほど細かく、テキストが鮮明。
～92 PPI - 粗い。ゲーミング用。
110 PPI - 標準的。バランス型。
163 PPI 以上 - 緻密。テキスト作業向け。
