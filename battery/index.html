    用途別早見表

    テレビ・エアコンのリモコン

    単3 / 単4

    家庭用リモコンでは単3か単4が多く使われます。電池ふたの内側の表記を確認してください。

    時計

    単3 / 単4 / SR系ボタン電池

    壁掛け時計は単3、小型時計や腕時計はボタン電池が多いです。

    車のスマートキー

    CR2032 / CR2025 など

    車種によって型番が異なります。交換前に必ず本体内の型番を確認してください。

    懐中電灯・防災用品

    単1 / 単2 / 単3

    大型ライトでは単1や単2、小型ライトでは単3や単4が使われます。

    体重計・キッチンスケール

    単4 / CR2032 など

    薄型の機器ではコイン電池、やや大型の機器では単4が多く使われます。

    カメラ周辺機器

    単3 / CR2 / CR123A / SR44 など

    ストロボ、露出計、古いカメラアクセサリーでは専用電池が使われる場合があります。

    電池サイズの簡易図

    乾電池のサイズ感

    単1が最も大きく、単4は細く小さいサイズです。9V形は角型です。

    コイン電池の見方

    CR2032は「直径20mm・厚さ3.2mm」という意味で読むと覚えやすいです。

    単3と単4の違い

    単4は単3より細く短い電池です。見た目が似ているので交換時は表記を確認します。

    電池交換時は、電池の向き・型番・電圧・充電可否を確認してください。異なる種類の電池を混ぜて使うのは避けてください。
